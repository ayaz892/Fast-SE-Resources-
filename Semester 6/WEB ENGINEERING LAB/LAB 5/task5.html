This is an example of a table layout using the Bootstrap framework. The table is created using the standard HTML `table` element, while the container, grid, and flex classes provided by Bootstrap are used to style the layout. The `container` class is used to provide a responsive, fixed-width container for the table, while the `row` and `col-lg-6` classes are used to create a two-column layout on larger screens. Finally, the `d-flex`, `align-items-center`, and `justify-content-center` classes are used to center the text in the right-hand column.
